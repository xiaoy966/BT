##安装过程
###需要先给termuxAPP所有文件访问权限
## termux 输入以下命令
#⚠️不管你之前有没有执行过以下命令，现在都必须执行
```
(以下步骤建议打开VPN执行命令，否则无法下载BT文件)
```
```
apt upgrade
```
```
pkg install git
```
```
rm -rf $HOME/BT
```
```
git clone https://github.com/xiaoy966/BT.git
```
```
(到这里请关闭VPN，否则登录不进去自己负责)
```
```
cd BT
```
```
chmod +x BT
```
```
bash BT
```
```
输入 2 并回车 进行搭建环境 然后输入 1 启动工具
```
## Start code
```
cd BT
```
```
./BT
```
