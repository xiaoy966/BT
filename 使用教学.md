##使用前请手动创建文件夹
##在termux运行目录 'BT/'内创建下面两个文件夹
```
需要修改的美化dat
```
```
修改后的美化dat
```
##在 '/storage/emulated/0/Download/' 路径创建下面三个文件夹
```
Paks
```
```
uexp解包
```
```
uexp打包
```
##如何一键制作美化
```
先把需要修改的dat文件放入 '需要修改的美化dat' 文件夹
```
```
然后在 'xynbxm/全自动制作美化配置.py' 文件内进行一键修改美化配置代码
```
```
打开termux 输入 'cd BT' 回车 然后输入'.bash BT' 或者 './BT' 回车
```
```
打开工具后: 输入1 回车，再次输入1 回车
```
```
然后等跑完代码之后，修改后的文件存放在 '需要修改的美化dat' 文件夹内
```
##如何打包
```
先把修改过的dat文件放入 '修改后的美化dat' 文件夹
```
```
然后把未经过任何修改的Pak文件放到 './' 目录
```
```
打开termux 输入 'cd BT' 回车 然后输入'bash BT' 或者 './BT' 回车
```
```
打开工具后: 输入2 回车，再次输入1 回车
```
```
等他跑完代码既可
```
```
打包好的文件在 './' 目录
```
##如何解包
```
先将需要解包的 Pak 文件放入 './' 目录
```
```
 '修改过的美化dat' 文件夹一定要没有任何文件
```
```
打开termux 输入 'cd BT' 回车 然后输入 'bash BT' 或者 './BT' 回车
```
```
打开工具后 输入 3 回车 ，再次输入 1 回车
```
```
等他跑完代码既可
```
```
解包好的文件在 '修改过的美化dat' 文件夹
```
##如何Uexp打包Pak
```
将未修改过的Pak文件放到 '/storage/emulated/0/Download/Paks/' 文件夹内
```
```
将修改后的uexp文件放入到 '/storage/emulated/0/Download/uexp打包/' 文件夹内
```
```
打开termux 输入 'cd BT' 回车 然后输入 'bash BT' 或者 './BT' 回车
```
```
打开工具后 输入 4 回车 ，再次输入 1 回车
```
```
等他跑完代码既可
```
```
打包好的文件在 '/storage/emulated/0/Download/Paks/' 文件夹
```
##如何Pak解包Uexp
```
将未修改过的Pak文件放到 '/storage/emulated/0/Download/Paks/' 文件夹内
```
```
打开termux 输入 'cd BT' 回车 然后输入 'bash BT' 或者 './BT' 回车
```
```
打开工具后 输入 5 回车 ，再次输入 1 回车
```
```
等他跑完代码既可
```
```
解包好的文件在 '/storage/emulated/0/Download/uexp解包/' 文件夹
```
